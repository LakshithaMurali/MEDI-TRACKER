# Real-Time Tracking and Medical Record App  

This project features a **real-time tracking device** integrated with a **mobile application**, ensuring real-time location tracking and offline access to critical medical records. The system is designed for emergency assistance and enhanced security, offering high reliability even in challenging conditions.

## üì± Mobile Application Features  

### **Key Functionality**  
1. **Real-Time Tracking**:  
   - Tracks the user‚Äôs location in real-time using GPS.  

2. **Medical Record Maintenance**:  
   - Stores vital medical information, accessible offline for emergencies.  

3. **Emergency Assistance**:  
   - Provides functionality to request help when the internet is unavailable.  

4. **QR Code Integration**:  
   - The tracking device is equipped with a QR code.  
   - Scanning the QR code redirects to the app for real-time location and medical records.
